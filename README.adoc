= Multi-Stage Container Builds with Hardened Images

This lab teaches the multi-stage container build pattern using Red Hat Hardened Images,
separating build-time dependencies from the minimal production runtime.

== Lab Overview

Participants build a production-ready Quarkus Java REST application using a two-stage
Containerfile: an openjdk builder stage for compilation, and a minimal runtime stage
for deployment.

== Modules

* Module 01: Examine the Application and Containerfile structure
* Module 02: Build the multi-stage image with Podman
* Module 03: Test REST endpoints and compare image sizes

== Duration

15-18 minutes
